Lab 15 - AJAX

Challenge:

For this lab, we learned how to use jQuery to make AJAX requests that run asynchronously. We chose to use the Google maps API and live data from the Santa Cruz map to show the live location of bus 15.

Problems:

I had some issues with the request to Santa Cruz Metro's site being blocked because it was cross-origin. I don't think there's a way around this issue when making requests from a webpage so I made a simple proxy server that would make the request for me, which allowed me to get the bus locations.
